# Huellas del Oeste - Refugio de Animales 🐾

Este es el proyecto institucional de **Huellas del Oeste**, una organización sin fines de lucro dedicada al rescate, rehabilitación y adopción responsable de perros en situación de calle en la zona oeste del Gran Buenos Aires.
